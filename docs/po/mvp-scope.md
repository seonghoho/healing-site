# MVP 범위 정리 (PO)

## 원칙
- 번아웃 사용자의 진입 장벽을 낮추기 위해, "짧고 확실한 완료 경험"을 최우선으로 둡니다.
- 최초 릴리스에서는 핵심 여정을 1분 내 완료 가능한 수준으로 제한합니다.

## Must (출시 필수)
- `/` 랜딩: 60초 호흡 가이드 + 시작 CTA
- `/today`: 오늘 미션 1개, 완료 체크, 칭찬 메시지, 누적 카운트
- `/history`: 최근 30일 기록(날짜/완료 여부)
- 로그인 없이 `localStorage` 저장 및 새로고침 유지
- 기본 레이아웃(홈/오늘/기록 네비게이션)
- 접근성 기본(포커스, aria-label, 텍스트 대비)

## Should (1차 고도화)
- 날짜 기반 deterministic 미션 선택(같은 날 동일 미션)
- 개발용 `resetAll()` 제공
- 문서-구현 매핑 유지

## Could (백로그)
- 주간 통계(연속 기록 일수)
- 미션 난이도/카테고리 필터
- 다국어 지원

## 핵심 여정
- `/` → `/today` → 완료 피드백 확인 → `/history` 확인
