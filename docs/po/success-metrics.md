# 성공 지표 (PO)

## KPI 1. 핵심 여정 기능 성공률
- 정의: `/` → `/today` → `/history` 흐름을 오류 없이 완료하는 비율
- 목표: QA 시나리오 기준 95% 이상
- 측정: 시나리오 테스트 통과율(수동 QA + 자동 테스트 로그)

## KPI 2. 저장 신뢰성
- 정의: 당일 완료 기록이 새로고침 이후에도 유지되는 비율
- 목표: 100%
- 측정: `localStorage` 기반 회귀 체크리스트 통과율

## KPI 3. 즉시 피드백 노출률
- 정의: 완료 클릭 직후 칭찬/피드백 UI가 노출되는 비율
- 목표: 100%
- 측정: UI 시나리오 점검(완료 액션 대비 피드백 렌더링)

## 운영 메모
- MVP 단계에서는 정교한 분석 도구보다 "재현 가능한 QA 로그"를 지표의 단일 기준으로 사용합니다.
