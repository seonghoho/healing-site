# 의사결정 요약 (PO)

## 반영된 결정
- D-001: 문서를 SSOT로 운영
- D-002: 하루 1개 미션 정책
- D-003: 저장소는 localStorage(v1)
- D-004: 최근 30일 기록 고정
- D-005: 테스트 전략(유틸 로직 중심 검증)

## 제품 방향에 미치는 영향
- 범위 통제: Must 중심으로 MVP 기능 고정
- 경험 일관성: 같은 날 동일 미션으로 학습 비용 최소화
- 구현 현실성: 서버 없이도 검증 가능한 데이터 구조 채택
- 확장 여지: 필터/통계/다국어는 백로그로 분리해 일정 리스크 축소
