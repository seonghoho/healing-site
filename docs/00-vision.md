# 00. Vision

## 타깃
- 번아웃으로 에너지가 낮아졌지만 짧은 자기돌봄 루틴부터 다시 시작하려는 사용자.

## 문제 정의 (1문장)
- 회복이 필요한 순간일수록 복잡한 습관 앱과 많은 목표가 심리적 부담을 높여 첫 행동을 막는다.

## 서비스 약속 (1문장)
- Healing Site는 매일 단 하나의 가벼운 미션과 즉시 성취 피드백으로 사용자가 1분 안에 회복 행동을 시작하고 마치게 돕는다.

## 핵심 가치 (3개)
1. 단순성: 하루 1개 미션만 제안해 결정 피로를 줄인다.
2. 즉시성: 완료 직후 칭찬과 상태 변화를 보여 성취감을 강화한다.
3. 비압박성: 강요 없는 문장과 속도 존중으로 지속 가능한 회복 리듬을 만든다.
