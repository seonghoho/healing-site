# 00. Vision

## 한 줄 비전
- 번아웃이 온 날에도 **60초 호흡 + 오늘의 미션 1개**로 회복 리듬을 다시 잇는 서비스.

## 타깃 사용자
- 업무와 일상에 지쳐 에너지 저하를 느끼는 번아웃 사용자
- 꾸준한 자기돌봄 루틴을 가볍게 시작하고 싶은 일반 사용자

## 문제 정의
- 회복이 필요할수록 긴 체크리스트와 강한 목표 문장이 부담으로 느껴진다.
- 기존 습관 앱은 미션 수가 많아 초기 이탈이 빠르게 발생한다.
- 당일 성취감이 즉시 보이지 않으면 재방문 동기가 낮아진다.

## 서비스 약속
- 한 번에 하나만 제안한다.
- 완수 직후 짧고 따뜻한 피드백으로 성취를 확인한다.
- 압박 문장을 배제하고 사용자의 현재 속도를 존중한다.

## MVP 우선순위 원칙
- **Must는 “오늘 바로 회복 행동을 완료”하게 만드는 최소 기능만 포함**한다.
- **Should는 품질/운영 효율을 높이되, Must 없이 단독 가치가 낮은 항목으로 제한**한다.
- **Could는 사용자 확장/개인화 기능으로 분리하여 일정 리스크를 차단**한다.
