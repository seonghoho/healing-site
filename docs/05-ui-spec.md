# 05. UI Spec (Text Wireframe)

## 공통 레이아웃
- 상단: 브랜드 + 네비게이션(홈/오늘/기록)
- 본문: 페이지 타이틀, 설명, 핵심 카드
- 스타일: 모바일 우선 단일 컬럼, 여백 중심 구성

## `/` 랜딩
1. Eyebrow: 회복 맥락 문구
2. H1: 60초 호흡 제안
3. 설명: 부담을 줄이는 안내 문장
4. 호흡 카드
- 가이드 문구(들이쉬기/내쉬기)
- 원형 애니메이션
- 남은 시간/진행률
- 시작/다시 버튼
5. CTA 버튼: `시작하기` -> `/today`

## `/today`
1. Eyebrow + 페이지 타이틀
2. 설명 문구(한 가지 미션 원칙)
3. 미션 카드
- 미션 제목/설명
- 예상 시간
- 완료 체크 버튼
- 칭찬 메시지(aria-live)
- 누적 완료 수
4. 개발 환경 한정 버튼: 전체 초기화

## `/history`
1. Eyebrow + 페이지 타이틀
2. 설명 문구(최근 30일 안내)
3. 기록 카드
- 날짜
- 완료/미완료 배지
- 완료 시 미션 제목
