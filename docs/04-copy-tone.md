# 04. Copy Tone

## 1) 말투 규칙
- 짧은 문장: 한 문장에 한 메시지만 담고, 문장 길이는 10~24자 중심으로 유지한다.
- 존댓말 유지: 기본 어미는 `-해요 / -했어요 / -할 수 있어요`로 통일한다.
- 압박 금지: 지시형 표현보다 선택형 표현을 사용한다. (`지금 해보세요` 대신 `지금 해볼까요`)
- 평가 금지: 사람을 평가하지 않고 행동과 상태만 묘사한다. (`잘했어요`는 가능, `성실하네요`는 지양)
- 회복 중심: 성과보다 호흡, 휴식, 재시작의 가치를 강조한다.

## 2) 금지어 / 권장어

| 구분 | 금지어/표현 | 권장어/대체 표현 |
| --- | --- | --- |
| 강요 | 반드시, 무조건, 지금 당장 | 천천히, 가능하면, 지금부터 |
| 비난 | 왜 못했나요, 또 실패했네요 | 아직 완료 전이에요, 다음에 다시 해봐요 |
| 낙인 | 게으름, 의지 부족 | 피곤한 날도 있어요, 쉬어도 괜찮아요 |
| 압박 | 늦었어요, 미뤘네요 | 오늘 안에 한 번이면 충분해요 |
| 단정 | 당신은 항상/절대 | 오늘은, 지금은, 이번에는 |

## 3) 상태별 톤 가이드

### 칭찬 문구
- 원칙: 결과 과시보다 "지금 한 행동"을 인정한다.
- 원칙: 과장된 감탄사(`대단해요!!!`)를 피하고 차분하게 마무리한다.
- 예시
  - "좋아요. 오늘 한 번을 해냈어요."
  - "기록했어요. 작은 루틴이 쌓이고 있어요."
  - "지금처럼 한 가지씩 하면 충분해요."

### 실패/미완료 문구
- 원칙: `실패`라는 단어를 쓰지 않는다.
- 원칙: 재시도 가능성과 선택권을 함께 제공한다.
- 예시
  - "아직 완료 전이에요. 지금 1분만 해볼까요?"
  - "오늘은 넘어가도 괜찮아요. 내일 다시 시작해요."
  - "기록되지 않았어요. 다시 눌러서 저장할 수 있어요."

### 빈 상태 문구
- 원칙: 비어 있음 자체를 문제로 보지 않는다.
- 원칙: 첫 행동 하나를 제안하되 강요하지 않는다.
- 예시
  - "아직 기록이 없어요. 오늘 한 번부터 시작해요."
  - "선택된 미션이 없어요. 부담 없는 미션 하나를 골라요."
  - "최근 30일 기록이 없어요. 오늘의 체크가 첫 기록이 돼요."

## 4) 금지 톤 요약
- 비교 유도 톤 금지: 타인, 지난 성과와의 비교 문구 사용 금지
- 죄책감 유도 톤 금지: 미완료를 도덕적 문제처럼 표현 금지
- 과장 칭찬 톤 금지: 현실감 없는 보상형 문구 사용 금지
