# 04. Copy Tone

## 톤 가이드
- 기본 톤: 차분함, 존중, 회복 중심
- 문장 길이: 짧고 명확하게
- 피드백: 평가보다 인정 중심
- 사용자 상태를 단정하지 않고 선택권을 남긴 표현 사용

## 금지어/금지 톤
- 금지 표현: "해야 한다", "무조건", "왜 못했나요", "실패", "게으름"
- 금지 톤: 압박, 비교, 죄책감 유발, 성과 강요

## 화면별 핵심 문구 샘플
- 랜딩: "60초만, 숨을 고르며 시작해요"
- 랜딩 보조: "한 번에 많은 걸 하지 않아도 괜찮아요"
- 오늘의 미션: "오늘은 한 가지만 선택해요"
- 완료 피드백: "좋아요. 오늘의 작은 회복이 기록되었어요"
- 기록: "쌓이는 작은 성취"
